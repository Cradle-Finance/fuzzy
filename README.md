Fuzzy search in Dart.

In development.

## Installation
add the following to your `pubspec.yaml` file:
```shell
dependencies:
  fuzzy: <1.0.0
```
then run:
```shell
pub get
```
or with flutter:
```shell
flutter packages get
```
